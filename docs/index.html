How it works

Discovery, not hosting

Karma-Ego does not host or redistribute any dataset. We maintain structured metadata so researchers and engineers can find the right data and contact maintainers directly.

01
Search the registry

Filter datasets by task type, sensor modality, volume, license, or institution.

02
Review structured metadata

Each entry includes format specs, annotation details, paper citation, and access instructions.

03
Contact maintainers directly

Every listing links to the original project page and maintainer contact. No middleman, no friction.

04
Contribute via pull request

Add a dataset by submitting a structured YAML entry to our GitHub repository.

Contribute

Add your dataset

If you maintain an egocentric or first-person video dataset, submit a pull request. Listing is free — gain visibility with researchers and robotics companies actively building physical AI systems.

Entry format

Each dataset is a YAML file in /registry/datasets/ 

name: Your Dataset Name
institution: University / Research Lab / Company
dataset_page: https://your-dataset-page.com
volume_hours: 120
modalities: [rgb, depth, imu, gaze]
license: CC BY 4.0
paper_url: https://arxiv.org/abs/xxxx.xxxxx
access_url: https://your-dataset-page.com
contact: maintainer@institution.edu
country: Your Country
task_type: [task name, activity type]
data_type: egocentric video
publisher: https://your-publisher-url.com

About

Why Karma-Ego

The physical AI research community lacks a single structured source of truth for egocentric video data. Karma-Ego is India's contribution to fixing that.

The Problem

Robotics researchers spend weeks finding relevant datasets — scattered across lab pages, arXiv, and institutional portals with no common structure.

The Solution

A community-maintained open registry. Structured metadata, direct maintainer contacts, and a searchable index — all in one GitHub repository.

Open by Design

Karma-Ego is fully open source. No sign-up, no API keys, no paywalls. Fork it, extend it, build on it.

India Initiative

Built in Bengaluru as India's contribution to the global physical AI and embodied intelligence research ecosystem.
